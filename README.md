# SeSAC AI 부트캠프 학습 기록 (노베이스)

비전공자/노베이스로 시작해서, 부트캠프 과제와 실습을 따라가며 기록합니다.  
목표는 “끝까지 따라가기 + 이해하고 정리하기” 입니다.

---

## 1) 내 상태 (현재 레벨)
- 전공: 영어영문
- 코딩 경험: 거의 없음
- 현재 우선순위: 환경 세팅 → 과제 완주 → 개념 조금씩 이해

---

## 2) 진행 중인 커리큘럼
- AI Literacy
- 자료구조/알고리즘 
- 정렬, 트리/그래프, 힙 
- 개발환경 구성

---

## 3) 내가 자주 막히는 포인트
- 터미널/명령어가 익숙하지 않음
- 가상환경(venv) 활성화 / 경로 문제
- 디버깅(브레이크포인트, step over/into/out)

---

## 4) 기록 규칙
- “완벽한 이해”보다 “완주” 우선
- 에러가 나면 에러 메시지 + 해결 과정을 같이 남기기
